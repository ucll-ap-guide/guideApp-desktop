GuidO guide

Creating a map

To create a map, you have the following options:

  1. Create a new map

    Create a new map given a name. If you pick a name that is already present on the server and proceed to upload it, the map on the server will be overridden.

  2. Get map from computer

    Choose a previously created map, stored on your PC, to edit.

  3. Get map from server

    Download an existing map from our server and edit it.

Editing a map

Creating

  1. Triangles, squares, pentagons

    Create a triangle, square or pentagon with a given name, description and color.

  2. Polygons

    Create a polygon shape given a name, a description, color and the amount of vertices desired. You can add a vertice to a polygon by using ctrl + double click on the polygon, near the closest vertice where you want to add an extra vertice. In delete mode (see later), this shortcut will instead delete the nearest vertice of the clicked polygon

  3. Emergency exit/door

    Create an emergency exit or a door given their name, door height, door width and its color. You can rotate doors by right clicking them.

  4. Node

    Create a node which can be placed to indicate a passable location. This is used in later indicating possible routes a visitor may take on your property.

  5. Point of interest

    Create an indicator for a number of locations which may be interesting to highlight for a vistor. Vending machines, toilets and such are all examples of points of interest.

  6. Label

    Create a label to be placed on the map to divulge extra information about the layout, structure or property in general that may not be expressed through any of the previous options.

Updating

When clicking the pencil icon on the taskbar (dark blue bar), you enter "edit mode". In edit mode, some figures such as rooms, doors, nodes and labels can be edited to change their attributes. To edit a figure, click it in edit mode and a dialog box will show up with the previous values.

Deleting

When clicking the eraser icon on the taskbar (dark blue bar), you enter "delete mode". In delete mode, any figure you click will be deleted from the map.

Connecting nodes

Enabling connecting mode

To start connecting the previously placed doors and nodes, you can click on the Connect Nodes button, located on the taskbar

Connecting nodes

Keep these things in mind when connecting nodes:

  1. Double click on a node, door or point of interest

    By double clicking on a connectable node, a popup will show asking to indicate the neighbors it connects to. You should enter 1 neighbor's id (indicated by white labels) per input field and use the checkbox to indicate whether or not you can travel both from and to from this point to its neighbor.

  2. Pathfinding

    There are 2 ways nodes can be linked, unidirectional and bidirectional. Unidirectional links are indicated by an orange arrow and bidirectional links are indicated by green lines. In the mobile app, these links will be used to be able to calculate the fastest way from any room or door to any other room or door. This is why it is important to give clear names to doors and points of interest, as they will be used as end destination or starting location.

Saving and sharing your work

Saving your work

You can save your work using the Save file on pc option, located on the taskbar. This will download the map to your PC in JSON format.

Sharing your work

You can share your work with the world by uploading it to the server using the Upload map option, located on the taskbar. This will upload your map to our server and make it available for all users.
